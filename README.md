# 🎮 game-web

Public web assets and pages for my games and game servers.

This repository is hosted using **GitHub Pages** and serves as the asset directory for:

**https://game.mrzsomi.top/**

The domain contains static files, web pages, and resources used by my game-related projects.

---

## 📦 Repository Contents

### 🌐 `index.html`

The main landing page for:

**https://game.mrzsomi.top/**

This page acts as the directory homepage for all game-related assets hosted in this repository.

It provides information about the purpose of this domain and quick access to related services.

Includes:

- Link to the main website
- Link to the server status page
- Information about this asset repository
- Directories that are available and are there.

---

### 📢 `motd/index.html`

Counter-Strike: Source server MOTD (Message of the Day) page.

Displayed to players when joining the server.

Features:

- Server information
- Rules
- Technical details
- Live server data

Live information is automatically updated using GitHub Actions.

---

### ⚙️ `.github/workflows/`

Contains automation workflows used by this repository.

Current automation:

- Fetches live game server information
- Updates generated data files
- Keeps server-related pages up to date
